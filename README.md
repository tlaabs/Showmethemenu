# 주요 기능
1. 거리 기반 검색  
위치 정보를 사용하여 30km 이내 한식 음식점을 검색할 수 있다.

2. 리뷰 기능  
외국인들이 음식점에대한 평가를 할 수 있다.

3. 메뉴 제공 기능  

4. NFC 태깅을 통한 메뉴 바로 보기  
매장내 테이블마다 부착된 NFC태그에 스마트폰을 태깅하면 메뉴를 바로 볼 수 있다.
